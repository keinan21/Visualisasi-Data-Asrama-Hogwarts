Ini adalah proyek visualisasi data interaktif yang dibuat untuk **tujuan pembelajaran Chart.js**. Menggunakan data fiktif yang terinspirasi dari dunia Harry Potter, dashboard ini menampilkan berbagai statistik mengenai asrama-asrama Hogwarts.

---

## Fitur Utama

* **Statistik Asrama:** Menampilkan jumlah murid, distribusi gender, dan sebaran murid per tahun ajaran untuk setiap asrama (Gryffindor, Hufflepuff, Ravenclaw, Slytherin).
* **Kinerja Akademik:** Membandingkan rata-rata nilai OWL dan NEWT antar asrama, serta jumlah murid berprestasi.
* **Nilai Mata Pelajaran:** Visualisasi perbandingan kekuatan asrama di berbagai mata pelajaran sihir menggunakan radar chart.
* **Perolehan Poin:** Menampilkan total perolehan poin asrama dalam periode waktu tertentu.
* **Desain Tema Harry Potter:** Seluruh visualisasi dan elemen halaman disesuaikan dengan palet warna dan nuansa khas dunia sihir Hogwarts.

---

## Tujuan Proyek

Proyek ini dibangun sebagai sarana untuk:

* Mempelajari dan menguasai berbagai jenis grafik yang ditawarkan oleh **Chart.js**.
* Memahami cara mengintegrasikan data JSON dengan Chart.js.
* Menerapkan penyesuaian gaya dan tema pada grafik dan halaman web.
* Mempraktikkan dasar-dasar pengembangan web frontend (HTML, CSS, JavaScript).

---

## Teknologi yang Digunakan

* **HTML5**
* **CSS3**
* **JavaScript**
* **Chart.js** (untuk visualisasi data)
* **Bootstrap 5** (untuk kerangka kerja UI)
